## Features

- [✓] Supports uploading and storing of images
- [✓] Supports returning GIFs answering search queries

<br>

## Dependencies

- Mongoose
- Express

## Running

First clone this repository

Then install all the dependencies

```bash
npm install
```

Then use node to start the webserver

```bash
node app.js
```
